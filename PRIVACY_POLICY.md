# Privacy Policy — JD Grab

_Last updated: 2026-06-30_

JD Grab is a Chrome extension that helps you select and copy job description
text on LinkedIn, Indeed, Glassdoor, Wellfound, and Welcome to the Jungle.

## Data Collection

JD Grab does **not** collect, transmit, or sell any personal data or browsing
activity. The extension does not contain any network/analytics code and makes
no external requests.

## Data Stored Locally

JD Grab uses the `chrome.storage.local` API to save your keyboard shortcut
preferences (e.g. the key combination you configure in Options) on your own
device. This data:

- Stays on your machine
- Is never transmitted anywhere
- Is only used to remember your shortcut settings between browser sessions

## Permissions Used

- **storage** — save your keyboard shortcut preferences locally.
- **tabs** — detect the active tab's URL so the extension only acts on
  supported job sites, and to open links per your configured shortcut
  behavior.
- **windows** — restore browser window focus reliably before performing text
  selection, across different site layouts.
- **Host permissions** (linkedin.com, indeed.com, glassdoor.com,
  wellfound.com, welcometothejungle.com) — required to run the content
  script that locates and selects job description text on these sites.
